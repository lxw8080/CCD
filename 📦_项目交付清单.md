## 🚀 部署清单

### 前置条件

- [x] Python 3.10+
- [x] Node.js 16+
- [x] PostgreSQL 12+
- [x] Django 4.2.7
- [x] Vue 3 + Vite

### 部署步骤

1. **后端部署**
   ```bash
   cd backend
   pip install -r requirements.txt
   python manage.py migrate
   python manage.py runserver
   ```

2. **前端部署**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **验证**
   - 访问 http://127.0.0.1:3000
   - 测试所有功能
